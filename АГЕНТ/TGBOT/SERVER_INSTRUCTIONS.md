# Инструкция по запуску бота на сервере (Ubuntu/Debian)

Если вы загрузили всю папку `АГЕНТ` на сервер, выполните следующие команды в консоли сервера.

### 1. Установка Node.js (если не установлен)
```bash
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt install -y nodejs
```

### 2. Установка PM2 (чтобы бот работал 24/7)
Это менеджер процессов, который перезапустит бота, если он упадет или сервер перезагрузится.
```bash
sudo npm install -g pm2
```

### 3. Установка зависимостей проекта
Перейдите в папку с проектом (замените путь на ваш) и установите библиотеки:
```bash
cd /путь/до/папки/АГЕНТ
npm install
```

### 4. Запуск бота через PM2
Мы запускаем бота из папки `TGBOT`, но находясь в корне `АГЕНТ` (чтобы подхватились настройки модулей из package.json).
```bash
pm2 start TGBOT/bot.js --name "trader-bot"
```

### 5. Полезные команды
- **Посмотреть лог (что пишет бот):**
  ```bash
  pm2 logs trader-bot
  ```
- **Остановить бота:**
  ```bash
  pm2 stop trader-bot
  ```
- **Перезапустить бота:**
  ```bash
  pm2 restart trader-bot
  ```
- **Сохранить список процессов (чтобы запускался после перезагрузки сервера):**
  ```bash
  pm2 save
  pm2 startup
  ```
