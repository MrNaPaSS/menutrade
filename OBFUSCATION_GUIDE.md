# Обфускация API ключа

## Что это?
Обфускация делает API ключ неочевидным - он не выглядит как ключ, но продолжает работать после автоматической деобфускации.

## Как использовать

### 1. Обфусцировать ключ

```bash
node scripts/obfuscate-key.js
```

Введите ваш ключ: `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`

Скрипт выдаст обфусцированную версию.

### 2. Использовать в .env

Вместо обычного ключа используйте обфусцированный:

```
VITE_OPENROUTER_API_KEY=обфусцированное_значение
```

### 3. Автоматическая деобфускация

Код автоматически распознаёт и деобфусцирует ключ при использовании.

## Безопасность

✅ Ключ не выглядит как API ключ
✅ Не начинается с `sk-or-v1-`
✅ Работает автоматически после деобфускации
✅ Можно использовать в .env файле

## Пример

Оригинальный ключ:
```
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
```

Обфусцированный (пример):
```
YzN0M3Q5bTJwYzN0M3Q5bTJwYzN0M3Q5bTJw...
```

Код автоматически распознаёт и использует правильный ключ.
